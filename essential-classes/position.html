Position

Position is another key property in CSS and Bootstrap has a full complement of classes for this. The position property in CSS lets you control how an element flows within the context of the existing page. Now Bootstrap allows you to control this by adding a position class with a series of values. And these are the values that you can use which are pretty much what you can do with CSS. Now static means that the element will flow with the natural flow of the page. This is the default. So things appear on the page in the order that you coded them. relative would mean that the element would go where it would normally go, but we can offset its position from that spot. Now with absolute the element is removed from where it would normally go and is positioned in relation to the nearest relatively positioned parent, or the window if nothing is positioned relatively before that. fixed means that the element will move relative to the viewport which is usually the browser window. And sticky means that the element will stick to a certain position as it scrolls by.
